# GitNut Localnet scripts

Scripts:
- `solana-validator.sh`  Start a local Solana validator and configure the local CLI.
- `airdrop.sh`           Airdrop SOL to a given pubkey (localnet).
- `deploy-programs.sh`   Build and deploy the GitNut Anchor program(s).

Tip:
Make scripts executable:
```bash
chmod +x configs/localnet/*.sh
```
