# Examples
These examples demonstrate how third-party users can integrate GitNut:

- minimal-static-site: publish a static site with GitNut manifest + attestation
- node-service: basic Node service with CI build + SBOM
- python-tool: a small CLI utility with reproducible build hints
- rust-cli: a simple Rust CLI with cargo-lock + SBOM hints
- monorepo: multi-package example (pnpm) with GitNut manifest
